## How to Use

1. **Browse**: Navigate through different quote sources (Quotes, Goals)
2. **Auto-Play**: Click the pause button or press spacebar to control auto-rotation
3. **Navigate**: Use Previous/Next buttons to go through your quote history
4. **Customize**: Add your own quotes to `public/quotes.txt` or biblical passages to `public/bible.txt`
